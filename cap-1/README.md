# Cap 1

En este capítulo sólo quiero mostrar como crear y compilar usando gcc.

## main.c

Para compilarlo si lo tienes todo bien instalado, solo debes abrir la consola y escribir:

### Windows

<pre><codes>
  gcc -o main.exe main.c
  main.exe
</codes></pre>

### Linux

<pre><codes>
  gcc -o main main.c
  ./main
</codes></pre>
